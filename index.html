A-Light Renderer Complex

A-Light is a persnal study project for Computer Graphic by Xingyu Chen using C#. It is a renderer complex project together with offline pathtracer and realtime rasterizer.

01.

ALight-Ray

ALight-Ray is a physically based pathtracing renderer independently developed by Xingyu Chen using C#.

  • Path Tracing
  • Motion blur
  • Bounding Volume Hierarchies
  • Texture Mapping
  • Normal Mapping
  • Reflectance Mapping
  • Perlin Noise
  • Light Material
  • 6-side Skybox
  • Model Rendering
  • Multiple Importance Sampling
  • Jitter sampling
  • Direct illumination
  • Scene Management
  • Multi-threaded rendering

02.

ALight-Raster

ALight-Raster is a realtime rasterizer renderer independently developed by Xingyu Chen using C#.

  • Programmable Rendering Pipeline
  • Vertex & Fragment Shader
  • Blin-Phong Shading
  • Midmapping
  • NormalMapping
  • Perspective Texture Mapping
  • Clipping in Homogenious coordinate
  • Model Rendering
  • Gramma Correction
  • HDR
  • Runtime script

03.

ALight-Creator

ALight-Creator is a desktop software to combine ALight-Ray and ALight-Raster together to make actual production.
